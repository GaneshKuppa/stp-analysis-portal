2 Correlation Analysis

Correlation matrices reveal relationships between attributes before factor extraction. Highly correlated attributes likely load on the same factor.

correlation_analysis.R
# Load brand rating data
brand.ratings <- read.csv("Data_Factor_Analysis.csv", stringsAsFactors = TRUE)

# Standardize all rating variables (critical step!)
brand.sc <- brand.ratings
brand.sc[, 1:9] <- scale(brand.ratings[, 1:9])

# Compute correlation matrix
cor_matrix <- cor(brand.sc[, 1:9])

# Visualize correlation structure
corrplot(cor_matrix, 
        method = "color", 
        type = "upper", 
        order = "hclust",
        tl.cex = 0.8,
        tl.col = "black",
        title = "Attribute Correlation Matrix",
        mar = c(0, 0, 2, 0))

# Identify highly correlated attribute pairs
findCorrelation(cor_matrix, cutoff = 0.7)
Attribute Correlation Heatmap
Colors represent correlation strength between attributes. Red = strong negative correlation, Blue = strong positive correlation. Clusters of similar colors reveal attribute groups that may form underlying factors.

3 Principal Component Analysis (PCA)

PCA reduces dimensionality by transforming correlated variables into uncorrelated principal components that capture maximum variance.

pca_analysis.R
# Aggregate to brand level (critical for perceptual maps!)
brand.mean <- aggregate(. ~ brand, data = brand.sc, mean)
rownames(brand.mean) <- brand.mean$brand
brand.mean$brand <- NULL

# Perform PCA using correlation matrix
brand.pc <- princomp(brand.mean, cor = TRUE)

# Scree plot to determine components
plot(brand.pc, type = "l", main = "PCA Scree Plot",
     xlab = "Principal Component", ylab = "Variance Explained")
abline(v = 2.5, col = "red", lty = 2)

# View component loadings (attribute contributions)
loadings(brand.pc)

# Create perceptual map (biplot)
biplot(brand.pc, 
       main = "Brand Positioning Perceptual Map",
       xlabs = rownames(brand.mean),
       col = c("darkblue", "darkred"))

4 Factor Analysis Implementation

Factor analysis identifies latent constructs (factors) that explain correlations among observed variables. Rotation (Varimax) simplifies interpretation.

factor_analysis.R
# Determine optimal number of factors
fa_scree <- nScree(brand.mean)
plot(fa_scree$Analysis$Eigenvalues, type = "b", pch = 19,
     xlab = "Factor Number", ylab = "Eigenvalue",
     main = "Factor Analysis: Scree Plot")
abline(h = 1, col = "red", lty = 2)  # Kaiser criterion

# Perform factor analysis with Varimax rotation
brand.fa <- factanal(brand.mean, 
                       factors = 2, 
                       rotation = "varimax", 
                       scores = "regression")

# Extract and interpret factor loadings
loadings_df <- as.data.frame(unclass(brand.fa$loadings[, 1:2]))
print(round(loadings_df, 3))

# Calculate communalities (variance explained per attribute)
communalities <- rowSums(loadings_df^2)
cbind(Loadings = loadings_df, Communality = round(communalities, 3))

# Extract brand positions (factor scores)
brand.scores <- brand.fa$scores
print(round(brand.scores, 3))
Rotation Methods Explained

Why rotate? Initial factor solutions often have complex structure (many variables loading moderately on multiple factors). Rotation simplifies interpretation by maximizing high loadings and minimizing low loadings.

Varimax rotation: Orthogonal rotation that maximizes variance of squared loadings within each factor. Creates "simple structure" where each variable loads strongly on only one factor.

Interpretation rule: |Œª| ‚â• 0.70 = strong loading; |Œª| 0.40-0.69 = moderate; |Œª| < 0.30 = negligible (consider dropping)

5 Creating Professional Perceptual Maps

Perceptual maps visualize brand positions relative to competitors and attribute directions. This reveals positioning opportunities and competitive threats.

perceptual_map.R
library(ggplot2)

# Prepare data for plotting
brand_positions <- as.data.frame(brand.scores)
brand_positions$brand <- rownames(brand_positions)

attribute_vectors <- as.data.frame(unclass(brand.fa$loadings[, 1:2]))
attribute_vectors$attribute <- rownames(attribute_vectors)

# Calculate variance explained
var_explained <- summary(brand.pc)$importance[2, 1:2] * 100

# Create professional perceptual map
ggplot() +
  # Brand points
  geom_point(data = brand_positions, 
            aes(x = Factor1, y = Factor2, color = brand), 
            size = 4, alpha = 0.85) +
  geom_text(data = brand_positions, 
           aes(x = Factor1, y = Factor2, label = brand), 
           hjust = "left", vjust = "bottom", size = 5, fontface = "bold") +
  
  # Attribute vectors
  geom_segment(data = attribute_vectors, 
             aes(x = 0, y = 0, 
                 xend = Factor1 * 2.5, yend = Factor2 * 2.5),
             arrow = arrow(length = unit(0.2, "cm")), 
             color = "darkblue", size = 1.2) +
  geom_text(data = attribute_vectors, 
           aes(x = Factor1 * 2.7, y = Factor2 * 2.7, label = attribute),
           color = "darkblue", size = 4, fontface = "italic") +
  
  # Quadrant lines
  geom_vline(xintercept = 0, linetype = "dashed", color = "gray50") +
  geom_hline(yintercept = 0, linetype = "dashed", color = "gray50") +
  
  # Labels and theme
  labs(title = "Coffee Brand Perceptual Map",
       subtitle = sprintf("Based on Factor Analysis (Total variance: %.1f%%)", 
                         sum(var_explained)),
       x = sprintf("Factor 1: Product Leadership (%.1f%%)", var_explained[1]),
       y = sprintf("Factor 2: Value Orientation (%.1f%%)", var_explained[2])) +
  theme_minimal(base_size = 14) +
  theme(legend.position = "none",
        plot.title = element_text(hjust = 0.5, face = "bold", size = 16),
        plot.subtitle = element_text(hjust = 0.5))
Professional Perceptual Map
Brands positioned in 2D space defined by underlying factors. Attribute vectors show direction of perceptions. Empty quadrants reveal positioning opportunities. Brand e (center) lacks differentiation.
üí° Strategic insight: Brands b and c dominate "Premium Leaders" quadrant. Brands f and g own "Value Brands" space. Upper-right quadrant represents untapped "Premium Value" opportunity.

Perceptual Map Interpretation Framework

  • Identify empty spaces: Gaps between brands represent unmet market needs and positioning opportunities
  • Assess competitive proximity: Brands close together compete directly for the same customers
  • Evaluate brand distinctiveness: Central brands lack differentiation; peripheral brands are distinctive
  • Map attribute directions: Shows what drives perceptions along each dimension
  • Strategic implications: Move brand toward desirable attributes; target underserved quadrants; differentiate from closest competitors
