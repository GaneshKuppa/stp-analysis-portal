2 STP Analysis R Cheatsheet

Quick reference guide for essential R commands used throughout the STP workflow:

STP Analysis in R: Quick Reference

Data Preparation

  • read.csv("file.csv") - Load CSV data
  • scale(df) - Standardize variables
  • ifelse(condition, yes, no) - Recode variables
  • str(df) - View data structure
  • summary(df) - Basic statistics

Segmentation

  • dist(df, method="euclidean") - Distance matrix
  • hclust(dist, method="complete") - Hierarchical clustering
  • cutree(hc, k=4) - Extract clusters
  • kmeans(df, centers=4) - K-means clustering
  • Mclust(df) - Model-based clustering

Positioning

  • cor(df) - Correlation matrix
  • corrplot(cor_mat) - Visualize correlations
  • princomp(df, cor=TRUE) - PCA
  • factanal(df, factors=2) - Factor analysis
  • aggregate(. ~ brand, data, mean) - Brand-level aggregation

Visualization

  • plot(hc) - Dendrogram
  • clusplot(df, clusters) - Cluster plot
  • biplot(pca) - PCA biplot
  • ggplot() + geom_point() - Custom perceptual maps
  • png("file.png"); plot(); dev.off() - Save plots
