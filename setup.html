R Environment Setup

Configure your R environment for success. Install required packages, prepare datasets, and master essential R objects for analytics workflows.

1 Installation Guide

R & RStudio Setup Checklist
  • Download and install R from CRAN
  • Download and install RStudio Desktop from RStudio website
  • Create a dedicated project folder named "STP_Analysis"
  • Set your working directory: Session > Set Working Directory > To Source File Location
  • Create an R script file (File > New File > R Script) and save as "STP_Workflow.R"

Setting Your Working Directory

The working directory is where R looks for files and saves outputs. Set it properly to avoid file path errors:

set_working_directory.R
# Method 1: Through RStudio menu
# Session > Set Working Directory > Choose Directory

# Method 2: Using R code (recommended for reproducibility)
setwd("C:/Users/YourName/Documents/STP_Analysis")

# Verify working directory
getwd()

2 Required R Packages

Install these essential packages for STP analysis. Run this code in your R console:

install_packages.R
# Install required packages (run once)
install.packages(c(
  "tidyverse",    # Data manipulation & visualization
  "cluster",      # Clustering algorithms
  "mclust",       # Model-based clustering
  "corrplot",     # Correlation visualizations
  "gplots",       # Heatmaps and visualizations
  "nFactors",     # Factor analysis tools
  "psych",        # Advanced factor analysis
  "factoextra"    # Enhanced factor visualizations
))

# Load libraries (run every session)
library(tidyverse)
library(cluster)
library(mclust)
library(corrplot)
library(gplots)
library(nFactors)
library(psych)
library(factoextra)
Pro Tip

After installing packages, restart RStudio to ensure all dependencies load correctly. Use sessionInfo() to verify package versions if you encounter errors.

Critical Preprocessing Step

Before analysis, you MUST standardize continuous variables using Z-scores. Variables on different scales (e.g., age 18-65 vs. income 20k-100k) will dominate distance calculations, producing meaningless clusters. Use scale() function in R.

4 R Objects Fundamentals

Vectors: One-Dimensional Collections

Vectors are the simplest R objects, containing elements of the same type:

vectors.R
# Numeric vector
numeric_vec <- c(1, 3.14, 5, 7)

# Logical vector
logical_vec <- c(TRUE, FALSE, TRUE)

# Character vector
char_vec <- c("sun", "care", "protection")

# Access elements (indexing starts at 1)
numeric_vec[2]  # Returns: 3.14
char_vec[1:2] # Returns: "sun" "care"

Data Frames: Rectangular Datasets

Data frames are the workhorse objects in R, used to hold datasets:

data_frames.R
# Create a data frame
customer_df <- data.frame(
  age = c(25, 34, 42, 28),
  income = c(45000, 62000, 78000, 51000),
  gender = factor(c("Female", "Male", "Female", "Male")),
  sunscreen_use = c(5, 2, 4, 3)  # 1-5 scale
)

# View structure
str(customer_df)  # Shows data types & dimensions
summary(customer_df)  # Basic statistics

# Access columns
customer_df$age  # Returns age column
customer_df[1, 3]   # Returns row 1, column 3 (gender of first customer)
Best Practice

Always convert categorical variables to factor type before analysis. This ensures R treats them appropriately in statistical models and visualizations.
