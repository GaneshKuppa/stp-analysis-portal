1 Segment Validation Exercise

Validate that your clusters are statistically meaningful and business-relevant.

segment_validation.R
# Load required libraries
library(dplyr)

# TASK 1: Statistical validation using ANOVA
# Test if income differs significantly across clusters
seg.df$cluster <- seg.mc$classification  # Using Mclust clusters
anova_result <- aov(income ~ factor(cluster), data = seg.df)
p_value <- summary(anova_result)[[1]][["Pr(>F)"]][1]

cat("ANOVA p-value for income across clusters:", round(p_value, 4), "\n")
if (p_value < 0.05) {
  cat("‚úÖ Clusters show statistically significant income differences\n")
} else {
  cat("‚ö†Ô∏è Clusters do NOT show significant income differences\n")
}

# TASK 2: Business relevance check
# Calculate subscription rate difference between top and bottom segments
segment_subs <- seg.df %>%
  group_by(cluster) %>%
  summarise(sub_rate = mean(subscribe) * 100) %>%
  arrange(desc(sub_rate))

top_segment_rate <- segment_subs$sub_rate[1]
bottom_segment_rate <- segment_subs$sub_rate[nrow(segment_subs)]
difference <- top_segment_rate - bottom_segment_rate

cat("\nSubscription rate difference between best and worst segments:", 
    round(difference, 1), "%\n")
if (difference > 25) {
  cat("‚úÖ Segments show strong business differentiation\n")
} else {
  cat("‚ö†Ô∏è Segments may not be sufficiently differentiated for targeting\n")
}

# TASK 3: Create segment profile visualization
library(ggplot2)

segment_profiles_long <- segment_profiles %>%
  select(segment, avg_income, subscribe_rate) %>%
  pivot_longer(cols = c(avg_income, subscribe_rate),
                 names_to = "metric", values_to = "value")

ggplot(segment_profiles_long, aes(x = factor(segment), y = value, fill = metric)) +
  geom_col(position = "dodge", width = 0.7) +
  labs(title = "Customer Segment Profiles",
       x = "Segment", y = "Value",
       caption = "Primary target: Segment with highest subscription rate") +
  theme_minimal(base_size = 14) +
  theme(legend.position = "top")

2 Positioning Gap Analysis Exercise

Identify the largest empty space in the perceptual map for strategic opportunity.

positioning_gap_analysis.R
# Load brand scores from factor analysis
brand_positions <- as.data.frame(brand.scores)

# TASK 1: Calculate centroid of all brands
centroid <- colMeans(brand_positions[, 1:2])

# TASK 2: Find most distinctive brand (farthest from centroid)
distances <- sqrt(rowSums((brand_positions[, 1:2] - centroid)^2))
most_distinctive <- names(which.max(distances))
max_distance <- max(distances)

cat("Most distinctive brand:", most_distinctive, 
    "(distance from center:", round(max_distance, 2), ")\n")

# TASK 3: Identify opposite quadrant (potential gap)
brand_pos <- brand_positions[most_distinctive, ]
opposite_quadrant <- ifelse(brand_pos[,1] > 0, "left", "right")
opposite_quadrant <- paste(opposite_quadrant, 
                            ifelse(brand_pos[,2] > 0, "bottom", "top"))

cat("Potential positioning gap:", opposite_quadrant, "quadrant\n")

# TASK 4: Calculate ideal position for new brand in gap
# (Midpoint between most distinctive brand and opposite corner)
opposite_corner <- c(
  ifelse(brand_pos[,1] > 0, -1, 1) * 2,
  ifelse(brand_pos[,2] > 0, -1, 1) * 2
)

ideal_position <- (brand_pos[,1:2] + opposite_corner) / 2

cat("\nRecommended position for new brand:\n")
cat("  Factor 1 (Product Leadership):", round(ideal_position[1], 2), "\n")
cat("  Factor 2 (Value Orientation):", round(ideal_position[2], 2), "\n")

# TASK 5: Determine attribute emphasis for new brand
# (Attributes with strongest loadings in gap direction)
gap_direction <- sign(ideal_position)
relevant_attributes <- rownames(subset(loadings_df, 
  (sign(Factor1) == gap_direction[1]) | 
  (sign(Factor2) == gap_direction[2])))

cat("\nRecommended attribute emphasis:\n")
for (attr in relevant_attributes) {
  cat("  -", attr, "\n")
}

3 Common Errors & Debugging Guide

Error Message Cause Solution
Error in dist(): non-numeric argument Factor variables not recoded to numeric before clustering Use ifelse() to convert factors to 0/1 before standardization
Error: cannot allocate vector... Dataset too large for memory Use kmeans() instead of hierarchical; sample data; increase RAM
All clusters have 1 observation k too large relative to N Reduce k; check scree plot/elbow method; verify data standardization
Correlation matrix not positive definite Too many variables, too few observations; multicollinearity Remove highly correlated variables (>0.9); increase sample size; use PCA first
Perceptual map looks random Forgot to aggregate to brand level before PCA/FA Use aggregate(. ~ brand, data, mean) before factor analysis
NAs introduced by coercion Non-numeric characters in numeric columns Check data with str(); clean data; use na.omit()
Debugging Workflow

When errors occur, follow this systematic approach:

  1. Read the error message carefully - R usually tells you exactly what's wrong
  2. Check data structure with str(your_data) and summary(your_data)
  3. Verify variable types - factors vs. numeric vs. character
  4. Check for missing values with sum(is.na(your_data))
  5. Test with small subset of data to isolate problem
  6. Search error message online - someone has likely solved it before
