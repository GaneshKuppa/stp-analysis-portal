# STP Analysis Learning Portal

A comprehensive, interactive learning portal for Segmentation, Targeting, and Positioning (STP) analysis using R.

## Features
- 9 modular HTML modules covering STP framework fundamentals to advanced implementation
- Complete R code examples with copy-paste functionality
- Responsive design works on all devices
- Zero external dependencies (100% self-contained)
- Production-ready for educational use

## Modules
1. Introduction to STP Framework
2. R Environment Setup
3. Segmentation Analysis
4. Positioning Analysis
5. Integrated STP Workflow
6. Practical Exercises
7. Master Implementation Program
8. Datasets & Resources

## Deployment
Hosted on Vercel: [https://stp-analysis-portal.vercel.app](https://stp-analysis-portal.vercel.app)
